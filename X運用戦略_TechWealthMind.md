## 2. コンテンツ戦略（投稿の黄金比）

### 投稿タイプの比率
| タイプ | 割合 | 目的 |
|--------|------|------|
| 知識・Tips共有（ブログ誘導なし） | 40% | フォロワー増加・信頼構築 |

| エンゲージメント系（質問・投票） | 20% | リプライ・拡散促進 |
| 自分の体験・ストーリー | 15% | 親近感・ファン化 |

### 投稿フォーマット別の効果

```

2/ ...
3/ ...
（続きはブログへ）

いいね or リプライで教えてください👇

```

40代時点の資産に〇〇円の差が生まれる。

その理由をブログで詳しく解説しました👇

```

#### 🟠 ストーリー型（ファン獲得）
```

今は月〇万円を安定して稼いでいます。

## 3. 投稿頻度と最適時間帯

### 推奨投稿頻度
- **理想**: 1日2〜3投稿
- **最低限**: 1日1投稿（継続性が最重要）

### 最適な投稿時間帯（日本時間）
| 時間帯 | 理由 |
|--------|------|
| **7:00〜8:00** | 通勤・朝の情報収集タイム |
| **12:00〜13:00** | ランチタイムのSNSチェック |
| **21:00〜23:00** | 夜のリラックスタイム（最もエンゲージが高い） |

### 曜日別の戦略
- **月曜**: モチベーション系・週の目標設定
- **火・水・木**: 知識・Tips・ブログ記事紹介（メインコンテンツ）
- **金曜**: 振り返り・まとめ系
- **土日**: 軽め・エンゲージメント重視（質問・投票）

### 使用するハッシュタグ（2〜3個に絞る）

## 5. フォロワーを増やす戦術

### フォロワー獲得の優先順位

1. **同ジャンルへのリプライ活動**（最も効果的）
   - フォロワー1,000〜10,000人の同テーマアカウントの投稿に、価値あるリプライを毎日5〜10件
   - 「同意します」ではなく「補足情報を添えたリプライ」が効果的

2. **スレッド形式の投稿**
   - 1ツイートより、5〜10ツイートのスレッドのほうが保存・リポスト率が高い
   - スレッドはアルゴリズムに優遇されやすい

3. **毎日決まった時間に投稿する**
   - Xのアルゴリズムは「アクティブなアカウント」を優遇する

4. **インフルエンサーの投稿にコメント**
   - 早期コメントはそのインフルエンサーのフォロワーに露出する機会になる

## 6. ブログへの誘導テクニック

### 基本方針：プロフィールリンクへの誘導に統一する
記事ごとにURLを用意する手間を省き、プロフィールに設定したブログリンクへ誘導する方針で運用する。Xはリンク付き投稿のリーチを下げる傾向があるため、この方針はリーチ面でも有利。

### NG：投稿本文に直接記事URLを貼る
```
❌ 「新記事書きました！ https://tech-wealth-mind.com/xxxx #ブログ」
→ Xはリンク付き投稿のリーチを下げる傾向がある
→ 記事ごとにURLを管理する手間がかかる
```

### OK：価値提供 → プロフィール誘導の流れ
```
✅ 本文で価値・興味を提供し、最後に「プロフィールのリンクからどうぞ👆」
→ リーチを最大化しつつ、ブログへ自然に誘導できる
```

### 効果的なCTA（Call to Action）
- 「詳しくはプロフィールのブログで解説しています👆」
- 「プロフィールのリンクから読めます👆」
- 「気になる方はプロフィールのブログをチェックしてみてください👆」
- 「プロフィールのリンクからどうぞ👆」
