<div class="lead-text"># 高配当株投資におすすめの証券口座3選【2024年版・現役投資家が比較】</div>


**結論から言います。高配当株投資を始めるなら、まずはSBI証券か楽天証券の一択です。** 特に理由がなければSBI証券をおすすめします。銘柄数・手数料・NISA対応のすべてでトップクラスだからです。

私は現在、SBI証券のNISA成長投資枠でイオン300株、三菱商事100株、KDDI200株など複数の高配当株を保有しています。正直イオン株は現在含み損-29.9%という悲惨な状況ですが……それはさておき、証券口座選びだけは後悔していません。新潟の地方サラリーマンとして、手数料や使い勝手をしっかり比較した上で選んだからです。

---

## SBI証券：高配当株投資の「総合力No.1」

| 項目 | 内容 |
|------|------|
| 国内株手数料 | 0円（ゼロ革命） |
| 取扱銘柄数 | 約4,000銘柄以上 |
| 新NISA対応 | 成長投資枠・つみたて枠ともに対応 |
| ポイント | Vポイント・Pontaポイントなど |

2023年秋から国内株の売買手数料が完全無料になりました。高配当株投資は「買ったら長期保有」が基本なので手数料の差は小さいように見えますが、銘柄を入れ替えるリバランス時には確実に効いてきます。

NISA成長投資枠の対応銘柄も非常に豊富で、私が保有している日経高配当50ETFや積水ハウスなども問題なく購入できます。三井住友カードでの積立投資でVポイントが貯まる点も地味に嬉しいポイントです。

**こんな人向け：** 銘柄をしっかり選びたい・ETFも組み合わせたい・ポイントも活用したい人

---

## 楽天証券：楽天経済圏ユーザーなら迷わずここ

| 項目 | 内容 |
|------|------|
| 国内株手数料 | 0円 |
| 取扱銘柄数 | 約4,000銘柄以上 |
| 新NISA対応 | 成長投資枠・つみたて枠ともに対応 |
| ポイント | 楽天ポイント |

楽天証券の最大の強みは**楽天ポイントとの連携**です。楽天カードで投信積立をすると最大1%のポイント還元があり、貯まったポイントで国内株を購入することもできます。楽天市場・楽天銀行・楽天カードをすでに使っているなら、証券口座もまとめてしまうのが効率的です。

投資情報ツール「マーケットスピード」も使いやすく、高配当株のスクリーニング（利回り・配当履歴での絞り込み）機能が充実しています。

**こんな人向け：** 楽天ユーザー・ポイントで株を買ってみたい人・スクリーニングを活用したい人

---

## 松井証券：サポートを重視するなら選択肢に

| 項目 | 内容 |
|------|------|
| 国内株手数料 | 0円（50万円以下） |
| 取扱銘柄数 | 約3,800銘柄 |
| 新NISA対応 | 成長投資枠・つみたて枠ともに対応 |
| サポート | 電話・チャット・週7日対応 |

松井証券は老舗のネット証券で、**カスタマーサポートの質の高さ**が際立ちます。週7日対応の電話サポートがあり、「高配当株を買おうとしたら操作がわからない」という初心者の悩みにも丁寧に応えてくれます。

銘柄数はSBI・楽天より若干少なめで、ポイント還元の仕組みも弱め。ただし、**「投資は初めてで不安が大きい」という方には精神的な安心感**という意味でおすすめできます。

**こんな人向け：** 初めての投資で不安・電話で質問しながら進めたい人

## 高配当株投資で証券口座を選ぶ際の重要ポイント

高配当株投資に特化して口座を選ぶなら、以下の3点を必ずチェックしてください。

**① 手数料が無料かどうか**
現在は主要ネット証券はほぼ無料です。逆に言えば、対面型の証券会社（野村・大和など）は手数料が高く、配当金の旨みが削られます。

**② NISA成長投資枠に対応しているか**
高配当株は成長投資枠で保有するのが鉄則。年間240万円まで非課税で運用でき、配当金にかかる約20%の税金が節約できます。私の場合、三菱商事の含み益+37.9%もNISA口座のおかげで非課税です。

**③ 銘柄の取り扱い数**
高配当株の候補は多岐にわたります。商社・通信・インフラ・リートなど幅広い業種から選ぶためにも、取扱銘柄が豊富なSBI・楽天が安心です。

## 初心者におすすめの口座開設手順【SBI証券の場合】

副業ゼロの会社員でも、口座開設は思ったより簡単です。

1. **SBI証券の公式サイトにアクセス**
2. 「口座開設」ボタンからメールアドレスを登録
3. 本人確認書類（マイナンバーカードが最速）をアップロード
4. **NISA口座も同時に申し込む**（これを忘れずに！）
5. 審査完了後（最短翌営業日）に入金して取引スタート

スマホだけで完結できます。マイナンバーカードがあれば最短で翌日には口座が使えます。

## 投稿者の所感

私自身はSBI証券一本で高配当株投資を続けていますが、楽天ユーザーなら楽天証券でも全く問題ないと思います。どちらかを選んでさっさと始めることが大事で、口座選びで悩んでいる時間が一番もったいないです。イオン株の含み損を抱えながらも配当金が振り込まれるたびに「やっていてよかった」と思えるのが高配当株投資の醍醐味なので、まずは一歩踏み出してみてください。
